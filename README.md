# Matrix Common Factor Finder
A lightweight program for finding common factors across matrices. This project analyzes matrix values and identifies the highest common factor available. The program also was created to test out my new numpy knowledge.
# Features
* Find common factors in matrices
* Fast and efficient calculations
* Simple command-line usage
